Detailný popis každého kroku

Krok 1: Zadaj svoju žiadosť 📝

Člen komunity (startup, študent, investor) pojde na verejný formulár a zadá svoju požiadavku:

  • Názov toho, čo potrebuješ
  • Detailný popis problému
  • Kategória (7 typov)
  • Email a telefón

Príklad:

Názov: Potrebujem React vývojára
Kategória: Hľadanie zamestnanca
Popis: Hľadám senior React vývojára s 5+ rokmi skúseností pre náš startupových tým. Full-time, remote.

Krok 2: Preverenie od správcu 🔍

Admin tím dostane žiadosť a môže ju preskúmať v admin paneli:

  • Čita úplné detaily žiadosti
  • Kontroluje korektnosť údajov
  • Nastavuje prioritu (kritická/vysoká/stredná/nízka)
  • Hodnota vplyvu (1-10)
  • Pridáva review poznámky

Admin nastaví:

  • Priorita: Vysoká
  • Kategória: Hiring
  • Vplyv: 9/10
  • Status: V preverovaní

Krok 3: Algoritmus nájde expertov 🧠

Keď admin označí žiadosť za "V preverovaní", systém automaticky vypočítava expertné zhodnosti:

  • Expertise Match: Skúsenosť (0-40 bodov)
  • Category Fit: Kategória (0-30 bodov)
  • Availability: Dostupnosť (0-20 bodov)
  • Workload: Pracovné zaťaženie (0-10 bodov)

Zhodnosti pre "React vývojár":

Anna Nováková

87% match

Krok 4: Admin priraďuje experta 📧

Admin vidí návrhy zhodností a vyberá, koho chce priradiť:

  • Vidí top expertov s koeficientom zhody
  • Čita info o expertovi (bio, skúsenosti)
  • Klikne na "Priradiť experta"
  • Expert dostane notifikáciu e-mailom

Expert dostane e-mail:

Ahoj Anna,

Máme pre teba novú príležitosť! Startup hľadá senior React vývojára.

Zaujímavé? Kontaktuj žiadateľa…

Krok 5: Expert pomaha 🤝

Expert a žiadateľ sa spájajú mimo systému. Expert:

  • Kontaktuje žiadateľa (e-mail)
  • Diskutuje o podrobnostiach
  • Poskytuje know-how a pomoc
  • Pomáha riešiť problém

Výsledok:

Expert pomôže so:

  • Výberom správneho kandidáta
  • Vyjednaním podmienok
  • Onboardingom
  • Technical poradenstvo

Krok 6: Tracking a výsledky 📊

Keď je problém vyriešený, admin označí žiadosť ako "Vyriešená":

  • Zaznamená výsledok
  • Pridáva potrebné poznámky
  • Nameraní sa uchová v systéme
  • Expert dostane kredit za pomoc

Výsledok v systéme:

Status: Vyriešená

Riešenie: Najatý senior React vývojár

Vplyv: ⭐⭐⭐⭐⭐ (5/5)

Expert: Anna Nováková

Súhrn procesu

Celý proces trvá od niekoľkých hodín do dní, v závislosti od komplexnosti požiadavky.
