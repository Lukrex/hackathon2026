O Community Help System

Systém navrhnutý pre HalovaMake Hackathon 2026

Čo je to?

Community Help System je digitálny systém na organizáciu a správu žiadostí o pomoc v rámci komunít (startupov, univerzít, VC ekosystémov). Rieši problém, že pomoc si často vyžaduje manuálne hľadanie a koordináciu, čím sa stávajú procesy chaosom v e-mailoch a správach.

Problém, ktorý riešime

  • Komunita nevie, kto potrebuje čo a kto môže pomôcť
  • Pomoc sa deje náhodne alebo iba manuálne
  • Nie je jasné, ktorá požiadavka má najvyššiu prioritu
  • Chýba prehľadnosť a koordinácia medzi členom tímu
  • Nemožné merať vplyv a výsledky pomoci

Naše riešenie

Vytvárame elegantný, webový systém, ktorý umožňuje:

  • Egynoduché podávanie: Komunita zadáva žiadosti verejným formulárom
  • Manuálne preverovanie: Admin tím skúma a kategorizuje požiadavky
  • Inteligentný matching: Systém automaticky navrhuje najvhodnejších expertov
  • Email notifikácie: Všetci sú informovaní prostredníctvom e-mailu
  • Tracking dopadom: Meríme výsledky a lepšujeme proces
  • Notion integrácia: Spojenie s existujúcimi nástrojmi tímu

Technologické pozadie

Backend

  • Django - Web framework
  • Django REST Framework - REST API
  • SQLite/PostgreSQL - Database
  • Celery - Task queue (optional)

Frontend

  • Django Templates - HTML rendering
  • React - Optional components
  • Pure CSS - Styling
  • Responsive Design - Mobile ready
